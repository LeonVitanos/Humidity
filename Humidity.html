Conclusions From Scatterplot matrix¶

The first thing i can see, comparing the temperature and apparent temperature graphs, is that the wind speed, wind bearing(direction) and humidity affect the apparent temperature:

-I would expect wind speed to have a higher effect on the apparent temperature (based on human knowledge). Also because there is a gap of values in the 'Apparent Temperature/Wind speed' scatterplot matrix (the one looking like pac-man), i think that the calculation of the apparent temperature is relied on an 'if this then that' rule, making it not so scientifically correct. 
-The effect of humidity is clearly visible, lower temperatures have higher humidity (negatively correlated)
-The effect of wind bearing is slightly visible.

The other thing i can notice, is that the temperature and the aparent temperature are about the same, but there are some slight differences in below ~+10C and above ~+30C. Specifically, below ~+10C, the apparent temperature is slightly lower than the temperature, and above ~+30C, the apparent temperature is slightly bigger than the temperature. Probably this has to do with how humans perceive very cold and very hot temperatures, or with wind speeds etc..

I also noticed that some data have pressure 0 while most of the data are between 900-1100. Wikipedia, says : "The record low is 870 mbar" so these are obviously false data, where the sensor failed to give a value. So i proceeded to a numeric analysis of the data.

Predict temperature only from humidity data¶

In [170]:
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error

# Remove a few rows that have 0 humidity (because that's not possible)
df = df[df['Humidity'] != 0]

# Split train/test set
train_set, test_set = train_test_split(df, test_size=0.2, random_state=34)
X_train = train_set['Humidity'].values.reshape(-1, 1)
y_train = train_set['Temperature (C)']
X_test = test_set['Humidity'].values.reshape(-1, 1)
y_test = test_set['Temperature (C)']

def train_and_predict(algorithm, verbose):
    model = algorithm
    model.fit(X_train, y_train)

    # Compute RMSE
    predictions = model.predict(X_test) # Predictions on the training set
    mse = mean_squared_error(y_test, predictions) # Computing MSE
    rmse = np.sqrt(mse) # Taking the root
    
    if verbose:
        print(rmse)
    else:
        return rmse
